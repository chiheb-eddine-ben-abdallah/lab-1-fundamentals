# Lab 1: React Fundamentals (Vite Edition)

Personal portfolio and React lab project built during the React Fundamentals course.

## Features & Completed Exercises

- Functional components & JSX basics
- Props and data passing
- Conditional rendering (Online/Offline StatusBadge)
- Component composition (BlogPost reusing UserCard)
- Lists with `.map()` (MovieList)
- Personal Portfolio Challenge (Header, About, Projects, Contact, Footer)

## Technologies

- React (with Vite)
- Pure CSS (minimal inline + index.css)
- No external UI libraries
